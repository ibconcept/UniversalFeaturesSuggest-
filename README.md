## âœ… What is UniversalFeaturesSuggest

**UniversalFeaturesSuggest (UFS)** is a **web-only, product-agnostic platform** that enables:

- Feature suggestions
- UI/UX improvement proposals
- Community voting & validation
- Feature bounties & rewards
- Direct routing to companies

> Think **GitHub Issues + Product Hunt + Bug Bounties**, but for **features**.

## ğŸ‘¤ User & Company Journey

### User Journey
1. Visit **UniversalFeaturesSuggest**
2. Select a product or company
3. Submit a feature or UI/UX idea
4. Post:
   - anonymously  
   - or with identity for reputation
5. Community discusses & upvotes
6. Feature reaches validation threshold
7. Company implements & rewards
8. User gains recognition, rank, or payout

### Company Journey
1. Create a public or private board
2. Receive structured, ranked ideas
3. Run feature or UI bounties
4. Track demand before building
5. Reward contributors
6. Discover power users & designers
